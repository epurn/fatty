# Development Goal

The repository should support autonomous development without lowering the quality bar.

Every agent run should be able to:

- find the right guidance quickly,
- work from a story or contract,
- make scoped code changes,
- update tests and docs,
- run verification,
- prepare a PR,
- hand off to a separate reviewer phase.

The root `AGENTS.md` stays lean. Detailed instructions live in focused playbooks and skills so agents only load what the task requires.
